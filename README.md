
# chatbot


## prepare
* node js
  * express module
    ```
    npm install express
    ```
  * nodejeiba (optimal)
  
* youtube API credentials
  * your_dir/.credentials/youtube-nodejs-quickstart.json

## usage
1. open your terminal, and type:
```
node main.js
```
2. open browser and enter url:
```
localhost:8081
```
(cause the default port is '8081')

3. chat with it !
